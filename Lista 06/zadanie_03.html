• Znaczniki css, html, text, attr, val w zależności od ilości wprowadzonych argumentów służą do zwracania lub ustawiania wybranej własności

    • css pozwala zmienić własności css w wybranym elemencie, podanie jednego argumentu (np.$('#element').css({'background-color'})) pobiera wartość koloru tła dla elementu o id 'element'. Przy użyciu dwóch argumentów (np. $('#element').css({'background-color',”blue”})) nastąpi ustawienie koloru tła dla elementu o id 'element' na niebieski.
    • html ustawia bądź zwraca zawartość danego elementu znajdującego się w ciele dokumentu HTML. Gdy nie podamy żadnego argumentu zwróci nam wartość elementu o podanym id, jeśli wprowadzimy jakąś treść jako argument funkcji zostanie ona umieszczona w wybranym elemencie.
    • text bez argumentu zwraca tekst znajdujący się w wybranym przez nas miejscu, natomiast z - ustawia treść dla tego elementu.
    • attr umożliwia ustawienie bądź zwrócenie atrybutu i jego wartości; pierwszy argument oznacza artybut z którego chcemy skorzystać, ewentualne umieszczenie drugiego argumentu ustawia mu konkretną wartość.
    • val zwraca lub ustawia wartość dla wybranego elementu. Metoda służy przede wszystkim do uzyskania wartości elementów formularza, takich jak input, select i textarea. Po wywołaniu na pustym elemencie zwraca undefined.

  • Elementy hide, remove, visibility, slideUp, fadeOut powodują zniknięcie danego elementu z ekranu, jednak każdy z nich robi to na inny sposób:
    • hide - ukrywa element, działa podobnie do własności CSS display: none.
    • remove - całkowicie usuwa element ze strony.
    • visibility - ukrywa element na stronie, od hide różni się tym, że ukrywa zawartość elementu zostawiając dla niego miejsce na stronie.
    • slideUp - przesuwa dany element w górę, ukrywając go na stronie. Nie będzie on już wpływał na układ strony.
    • fadeOut - ukrywa dany element nadając mu efekt powolnego blaknięcia, aż stanie się niewidoczny. Nie ma on już wpływu na układ strony.

  • Omów działanie metod:
    • before - wstawia coś (np. tekst) przed wybranym przez nas elementem, lub elementami (w przypadku wpisania $('p').before() dodatkowy element pojawi się przed każdym z paragrafów)

    • after - wstawia coś (np. tekst) po wybranym przez nas elemencie, lub elementach (w przypadku wpisania $('p').after() dodatkowy element pojawi się po każdym paragrafie)

    • prepend - umieszcza wybrany przez nas obiekt na początku danego elementu (nie przed!), przykładowo
      $("ol").prepend("< li> Prepended item < / li>");
      umieści na początku listy nowy element zawierający napis "prepended item".

    • append - umieszcza wybrany przez nas obiekt na końcu danego elementu, przykładowo
      $("ol").append("< li> Prepended item < / li>");
      umieści na końcu listy nowy element zawierający napis "prepended item".

    • remove - całkowicie usuwa dany element ze strony

    • empty - opróżnia element z jego dzieci, przykładowo jeśli umieścimy w skrypcie zapis
      $("#div1").empty();
      to ze strony znikną wszystkie obiekty znajdujące się we wnętrzu elementu o id "div1"

  • animate - zmienia styl CSS obiektu, a robiąc to stopniowo stwarza efekt animacji. Argumentem dla tej własności mogą być jedynie wartości liczbowe(height,margin), animate nie jest w stanie tworzyć np. przejść między kolorami.
